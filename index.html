DeluxeTags Configuration

DeluxeTags is a premium plugin for Minecraft servers that allows administrators to create custom tags for players. These tags can be assigned based on ranks, permissions, or through in-game purchases. The plugin supports RGB colors, custom formatting, animations, and seamless integration with chat plugins.

Key Features

  • Custom tag creation with RGB color support
  • Permission-based tag assignment
  • Tag animations and special effects
  • Seamless chat plugin integration
  • In-game GUI for players to select their tags
